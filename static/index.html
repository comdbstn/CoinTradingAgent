LLM 기반 자동 코드 수정 시스템

시스템 개요

이 시스템은 TradingView에서 보내는 웹훅을 수신하여 자동으로 Pine Script 코드를 분석하고 개선하는 기능을 제공합니다.

대규모 언어 모델(LLM)을 활용하여 거래 데이터와 전략 코드를 분석하고, 문제점을 파악하여 자동으로 코드를 수정합니다.

API 엔드포인트

TradingView 웹훅 수신

POST /webhook/

TradingView에서 보내는 웹훅을 수신하여 자동으로 전략 코드를 분석하고 개선합니다.

수정 기록 조회

GET /webhook/history

코드 수정 기록을 시간순으로 조회합니다.

테스트용 분석 실행

POST /webhook/test-analyze

테스트용 엔드포인트: 예제 전략 코드와 최근 웹훅 로그를 분석하여 수정된 코드를 생성합니다.
